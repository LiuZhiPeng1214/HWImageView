
# HWImageView[![](https://jitpack.io/v/LiuZhiPeng1214/HWImageView.svg)](https://jitpack.io/#LiuZhiPeng1214/HWImageView)
能够加载webp,gif,png等格式的图片框架,无任何学习成本，傻瓜式接入

## 接入方式
  - gradle
    在 project中根目录加入
   ```
   allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
   ```
